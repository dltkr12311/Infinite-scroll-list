# InfiniteScroll Refator

## 바뀔점
- 폴더 구조 변경 
- 스타일 밑으로 변경
- 로딩중 변경
- 맨 마지막 확인 후 종료
### 과제 구현 목록
- [x] List Item
- [x] infinite scroll 구현
- [x] data fetching

### 설치 및 실행
`npm install`명령어 입력 후 설치 하고 `npm start`명령어 입력 후 실행
